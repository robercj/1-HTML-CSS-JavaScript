      Embed an image file

      • Open the folder named 1-HTML-CSS-JavaScript in VS Code. 
      • Create a new subfolder named images. 
      • Download the ext-icon.png file in the Exercise Resources section above, and save it to this images folder.
      • Open the file install-extension.html.
      • In the first listitem of the ordered list in that page, insert an <img> element before the word "Extensions" and give it a src attribute with the value of "images/ext-icon.png". Give it an alt attribute with the value of "Extensions view icon". 


      View your HTML file

      • Use Live Server to open the install-extension.html page in your browser. 
      • Verify that the image shows up correctly.


      Embed a video file

      • In your 1-HTML-CSS-JavaScript folder, create another subfolder named videos. This should be at the same level as the images folder, not inside it. 
      • Download the video file from the Exercise Resources above (right click and Save link as...) and save it to the videos folder.
      • In the file install-extension.html, below the second </section> closing tag, add a new <section> element.
      • Inside this third section, add a <video> element with the src of "videos/install-extension.mp4". 
      • Give it the Boolean attributes controls and loop.
      • Add appropriate fallback text inside the <video> ... </video> tags.

      View your HTML file

      • Use Live Server to open the install-extension.html page in your browser. 
      • Test that you can see the mp4 video with controls, and that you can play it. Make sure that it loops when finished.

      Try it yourself

      • As an optional exercise, here's are two bonus challenges for you: 
        1. Add an <a> element in the <nav> element that, when clicked, takes you to the third <section> element. Use the word "Video" for the text content of the <a> element.
        2. Add another <img> element just below the closing </ul> tag inside the second <section> element.
            • For this image element's src, use this absolute path on a web server instead of a local file:

      "https://storage.googleapis.com/nucamp-production.appspot.com/learnfiles/open-with-live-server.png"
            • Figure out an appropriate text for the alt attribute.
